Biography

I’m an assistant researcher at the software engineering group (SEG) in Nanjing University. My research interests lie in software testing and program language, including static and dynamic analysis. Recently, I focus on improving the performance of program analysis by machine learning.

Tools

  • DATA-Dang is a lightweight approach to mitigate dangling pointers at runtime by inserting intermediate pointers between a pointer and an object that the pointer points to.

  • DATA-SAT is a CNN-based constraint prediction tool that accelerates constraint solving in symbolic execution.

  • DATA-GINN is a new graph neural architecture, which focuses exclusively on intervals (generally manifested in looping construct) for mining the feature representation of a program. Furthermore, GINN operates on a hierarchy of intervals for scaling the learning to large graphs.

  • DATA-SA is a two-stage (light-weight and heavy-weight) approach to resolving a paradox between precision and scalability in data race detection.

  • DATA-SSD is a tool that detects data races in interrupt-driven program. It uses a combination of static analysis, symbolic execution and dynamic simulation to synthesize the failure-inducing data input and interrupt interleaving schedule.

  • DATA-Dataset is a bug dataset based on defect4j, bugs-dot-jar, BugSwarm and the extended dataset from ye et. al [FSE'14].
